MonikaAfterStory是一免费游戏的mod。心跳文学部！ 由 Team Salvato 制作。 MAS 基于第三周目制作，创建了你与Monika的永恒生活， 特点是新的事件， handlers, 和更多的跳杀!

请在 这里 最新的稳定版本更新。

如果你想自己创造一个像这样的Mod，请见我们的姊妹项目：DDLCModTemplate.

安装流程

看MAS安装视频教程： https://www.bilibili.com/video/BV1u741177vP

  • 跳转到 发布更新的页面.

  • 点击最新版本的链接。 你将要下载一份zip文件。

  • 把所有文件拖放到 /game 文件夹，本文件夹在你的《心跳文学部》的安装目录中

  • 运行ddlc.exe，MonikaAfterStory的Mod将会自动安装。

注意: 从资源库中下载的源文件是为了开发目的放置的，如果用来修改游戏，可能不会像预期的那样生效。 请仅用 正式发布的版本。

如需更多安装帮助，请查看我们的 FAQ（Tieba）

新增内容到MAS

想在MAS中添加一些内容？下面是游戏中使用的重要.RPY文件列表。

  • script-ch30.rpy: MAS的主要流程。这就是idle的发生。这里不会翻译了（
  • script-topics.rpy: 所有的 随机 、 提问 话题都包括在内。 您可以通过查看下面的信息添加自己的对话!
  • script-greetings.rpy: 你进游戏时的欢迎语。
  • script-farewells.rpy: 你关游戏时的告别语。
  • script-moods.rpy: 告诉莫妮卡你的 心情 。
  • script-stories.rpy: 添加故事，让莫妮卡讲给你听。
  • script-compliments.rpy: 添加你可以对莫妮卡说的赞美的话。
  • script-apologies.rpy: 加入要道歉的事情。

如果你想在太空小房子中添加更多的对话，请导航到script-topics.rpy并使用此模板。

示例： `renpy init 5 python: addEvent( Event( persistent.eventdatabase, eventlabel="monikaexample", # event label (必须和其他的不一样！！！) category=["example", "topic"], # 此主题所属类别的列表（这些名称自动大写） prompt="Example Topic", # 按钮显示的文本 random=True, # 如果这个主题应该随机出现，则为True pool=True # 如果这个话题应该出现在 "提问 "中，则为真。 ) )

label monika_example: m 1a "这是一个示例话题。" m 3d "我觉得这其实并不属于这里......" m 2e "为什么有人会直接将示例模板添加到mod中？" m 5r "他们真的不应该再被允许向这个资料库贡献了。" return **关于Event的所有可能的关键字的完整解释和详细信息，请查看Event的文档，该文档位于 definitions.rpy`**

对于比简单对话更复杂的事情，请参考Ren'Py在线文档。
